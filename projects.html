AutoU Code Challenge

O projeto consiste em um classificador de emails, onde ele lê o email inserido (via texto, pdf o eml), aplicas tecnicas de préprocessamento, e a partir daí define uma categoria para o email, se é Produtivo ou Improdutivo, seguindo as diretrizes do desafio, e gera uma resposta automática fazendo uso da API do Gemini.






Karaoke

O karaoke começou como um projeto para estudar websockets e API's REST, o projeto consiste em um sistema de gerenciamento de músicas, onde é possível cadastrar, editar, deletar e visualizar a fila de músicas, integrado com o celular a partir de um endereço local fornecido pelo próprio sistema via QR Code.





Target Code Challenge

O projeto foi desenvolvido com um desafio tanto pessoal quanto profissional, o projeto consiste em um sistema de gerenciamento de tarefas, onde é possível cadastrar, editar, deletar e visualizar tarefas, além de possuir um sistema de cadastro com autenticação, também é possivel editar, deletar e visualizar usuários, tudo com um sistema de controle de acesso baseado em nível. O projeto foi desenvolvido com NodeJS, HTML, CSS, JavaScript e MongoDB, com o objetivo de ser escalável e de fácil manutenção, para adição/remoção posterior de funcionalidades.





Uber Code Challenge

O projeto foi desenvolvido com o proposito de treinamento e aprofundamento da tecnologia Springboot, o projeto consiste em um service para disparos de emails, utilizando aws-ses como serviço de envio de emails. Foi criado com arquitetura limpa, com o objetivo de ser escalável e de fácil manutenção, para adição/remoção posterior de provedores de emails.





DSList

O projeto foi desenvolvido como forma de estudar e familiarizar com Springboot, Docker, Postman e com o banco de dados PostgreSQL. O projeto consiste em uma API REST para gerenciamento de listas de jogos, onde é possível cadastrar, editar, deletar e visualizar as listas e seus jogos.





Grafos

O projeto foi desenvolvido para a disciplina de Algoritmos em Grafos, com o objetivo de analisar os aspectos de um dado grafo, como: Vetores de bipartição, número mínimo de cores para colorir, matriz de adjaçência, e matriz da árvore geradora.
