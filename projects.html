DSList

O projeto foi desenvolvido como forma de estudar e familiarizar com Springboot, Docker, Postman e com o banco de dados PostgreSQL. O projeto consiste em uma API REST para gerenciamento de listas de jogos, onde é possível cadastrar, editar, deletar e visualizar as listas e seus jogos.





Grafos

O projeto foi desenvolvido para a disciplina de Algoritmos em Grafos, com o objetivo de analisar os aspectos de um dado grafo, como: Vetores de bipartição, número mínimo de cores para colorir, matriz de adjaçência, e matriz da árvore geradora.
